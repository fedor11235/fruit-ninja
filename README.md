# Fruit Ninja

Аркада в духе Fruit Ninja на чистом JS + Canvas. Без сборки и зависимостей — готова к загрузке в [Яндекс Игры](https://yandex.ru/games/).

## Управление

- Режь фрукты свайпом (мышь или палец).
- Бомба 💣 — мгновенный проигрыш.
- Уронил фрукт — минус жизнь (всего 3).
- 3+ фрукта одним свайпом — комбо, очки x2.

## Локальный запуск

```bash
npx serve .
# или
python3 -m http.server 8000
```

Открыть `http://localhost:8000`. Без окружения Яндекса SDK не загрузится — игра это определяет и работает автономно.

## Сборка архива для Яндекс Игр

```bash
./build.sh
```

Скрипт соберёт `dist/fruit-ninja.zip` с `index.html` в корне архива — именно такой формат требует консоль разработчика Яндекс Игр.

## Интеграция с Yandex Games SDK

- SDK подключается из `https://sdk.games.s3.yandex.net/sdk.js`.
- После инициализации вызывается `ysdk.features.LoadingAPI.ready()`.
- Начало/конец геймплея размечены `GameplayAPI.start()/stop()` (включая паузу при сворачивании вкладки).
- Рекорд хранится в `localStorage` и дублируется в данные игрока через `player.setData()`.
